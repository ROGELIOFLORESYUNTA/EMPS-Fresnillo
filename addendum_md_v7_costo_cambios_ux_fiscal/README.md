# Addendum v7 - Costo de cambios, explicabilidad visual y parámetros vivos

Este paquete se agrega al sistema actual. No reemplaza lo que ya existe.

## Objetivo

Fortalecer EMPS-Fresnillo para que calcule y explique el costo real de una solicitud de cambio en proyectos municipales de software, incluyendo:

- artefactos afectados;
- fase del proyecto;
- claridad de la solicitud;
- modo de desarrollo;
- riesgo en datos, seguridad e integraciones;
- impacto en pruebas, capacitación y mantenimiento;
- costo fiscal-laboral y flujo de efectivo;
- explicación visual para Ayuntamiento y proveedor.

## Regla de integración

El sistema actual ya tiene ChangeRequest, ChangeImpactAssessment, API de cambios, parámetros, reportes y pantallas. Este addendum pide mejoras aditivas: wizard visual, explicabilidad, parámetros vivos y reportes. No se debe rehacer la arquitectura.

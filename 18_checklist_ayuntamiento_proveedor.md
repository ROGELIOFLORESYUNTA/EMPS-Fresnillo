# Checklist Ayuntamiento y proveedor

## Para Ayuntamiento

- El alcance está escrito por módulos.
- Hay criterios de aceptación.
- Se separa garantía de nuevo alcance.
- Existe mantenimiento posterior.
- El proveedor explica equipo y disponibilidad.
- La forma de pago no obliga a financiar todo al proveedor.
- La cotización no depende de una sola persona.
- Hay plan de respaldo y documentación.
- Se contemplan pruebas y capacitación.
- Se contempla seguridad y privacidad.

## Para proveedor

- El precio cubre nómina real.
- El precio cubre impuestos y cargas.
- El anticipo cubre arranque.
- Los pagos por entregables cubren flujo.
- El contrato define cambios.
- El mantenimiento está cotizado.
- El riesgo de salida de personal está cubierto.
- Las herramientas y licencias están consideradas.
- Hay margen suficiente.
- El proyecto no depende de trabajar informalmente.

# Validación de investigación

## Objetivo

Probar si el modelo integrado mejora la estimación temprana frente a una estimación simple de horas.

## Casos mínimos

- Caso simple.
- Caso mediano municipal.
- Caso con integración compleja.
- Caso con cambios de alcance.
- Caso con riesgo financiero.

## Variables a comparar

- Esfuerzo estimado.
- Tiempo estimado.
- Costo técnico.
- Costo fiscal-laboral.
- Capital de trabajo.
- Riesgo de cambios.
- Mantenimiento.

## Evidencia esperada

- Reportes generados.
- Versiones de estimación.
- Cambios registrados.
- Comparación de escenarios.
- Conclusiones para resultados y discusión del artículo.

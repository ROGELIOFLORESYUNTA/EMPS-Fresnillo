Estoy trabajando un subproyecto de análisis de datos para el artículo científico preliminar de EMPS-Fresnillo.

Contexto académico:
- Seminario de Investigación, Ingeniería en Software, UAZ, 2026.
- Tema: estimación temprana de esfuerzo, tiempo y costo en proyectos de software para gobiernos municipales.
- El sistema EMPS-Fresnillo ya existe como prototipo, pero todavía no tiene suficientes datos reales locales para validar la hipótesis.
- Por eso se usará un análisis preliminar con datasets públicos de ingeniería de software.

Objetivo de esta carpeta:
- Descargar/preparar JOSSE y SEERA.
- Ejecutar regresión y modelos de contraste.
- Generar métricas, tablas, figuras y fragmentos LaTeX.
- Producir un artículo con apariencia científica.

Reglas:
- No cambiar el sistema EMPS-Fresnillo.
- No inventar resultados.
- Si un dataset no descarga o no se reconocen columnas, reportarlo como limitación.
- Mantener lenguaje académico.
- Usar JOSSE como dataset principal y SEERA como contraste.
- Public Jira queda opcional por tamaño.

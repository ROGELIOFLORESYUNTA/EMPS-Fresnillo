# Cómo defender esta decisión ante la maestra

## Si pregunta: ¿con qué base de datos trabajaste?

Se trabajó con datasets públicos de ingeniería de software. El principal es JOSSE, que contiene tareas de desarrollo y mantenimiento con esfuerzo real y estimaciones expertas. Como contraste se usa SEERA, un dataset de proyectos de software en entornos restringidos.

## Si pregunta: ¿dónde está machine learning?

El análisis aplica modelos de regresión para estimar esfuerzo a partir de variables tempranas. La primera etapa usa modelos simples y explicables: línea base, regresión lineal y bosque aleatorio. El objetivo no es presumir un modelo complejo, sino demostrar que los datos tempranos pueden apoyar la estimación.

## Si pregunta: ¿qué tiene que ver con Fresnillo?

Los datasets públicos no sustituyen datos de Fresnillo. Se usan como evidencia preliminar porque todavía no existe un historial local suficiente. El prototipo EMPS-Fresnillo se propone como instrumento para capturar datos locales y validar la hipótesis en una etapa posterior.

## Si pregunta: ¿qué se va a comprobar?

Se busca comprobar de forma preliminar que una estimación basada solo en opinión o precio global es insuficiente. Las variables tempranas, el tipo de tarea, el cambio, el modo de desarrollo y el costo completo del proveedor deben registrarse para mejorar la estimación.

## Si pregunta: ¿por qué no usar directamente el sistema?

Porque el sistema todavía no tiene historial suficiente de proyectos reales terminados. Por eso se usa primero evidencia pública y después el sistema funcionará como base de captura local.

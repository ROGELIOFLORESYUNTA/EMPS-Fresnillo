# Criterios de aceptación

1. Se puede crear un proyecto completo.
2. Se pueden registrar módulos e historias.
3. El sistema calcula escenarios por modo de desarrollo.
4. Se pueden modificar parámetros fiscales.
5. El cálculo usa los parámetros guardados.
6. El sistema calcula costo de equipo.
7. El sistema calcula IVA, subtotal y total.
8. El sistema calcula flujo de efectivo.
9. El sistema registra cambios.
10. El sistema genera reportes.
11. El sistema guarda versión de estimación.
12. El sistema advierte si faltan datos.
13. El sistema no borra información histórica al recalcular.
14. El sistema explica los riesgos en lenguaje entendible.
